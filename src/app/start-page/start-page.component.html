Дипломный проект FullSteck


Проект создан по принципу Single Page Application.

В целях демонстрации в проекте применены различные способы реализации функционала сервиса.

По-умолчанию в проекте созданы два пользователя: admin с правами администратора и test с правами пользователя, которые имеют одноименные пароли. Администратору добавлена функция "Очистить базу", которая очищает всю базу данных и привоводит сервис в состояние "по-умолчанию". Вы можете зарегистрировать неограниченое количество пользователей с уникальными именами. Требования к имени пользователя и паролю можно посмотреть нажав на значек информации возле поля ввода при авторизации или регистрации. Переходы между стартовой и домащней страницами осуществляется по клику на логотип.

Ссылки на исходный код находяться в футтере страницы.
